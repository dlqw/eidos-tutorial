# Eidos language 0.5.0-alpha.4 capability-constrained build metaprogramming

- Documented the compiler-built-in `Build` domain and the strict separation between pure comptime and capability-bearing build programs.
- Added the `[build]` manifest contract for a build program, declared file/directory inputs, registered environment variables, disjoint output roots, and explicitly registered host tools.
- Documented typed `Build::Fs`, `Build::Env`, `Build::Process`, and `Build::Emit` capabilities plus canonical declarative `BuildGraph`, command steps, generated sources, artifacts, and dependency edges.
- Documented host/target separation, graph validation, deterministic topological execution, dependency/output hashing, BuildGraph caching, generated-source import roots, `--emit-build-graph`, and `--trace-build`.
- Added a verified minimal Build host tutorial project without opening filesystem, environment, process, network, or FFI access to ordinary pure comptime.
