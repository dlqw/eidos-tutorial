# Eidos 0.8.0-alpha.1：隐式 Unit body 与分支选择

- sync-id：`eidos-2026-07-23-implicit-unit-selection`
- 新增首个运行时参数为 `Unit` 时省略 `_ =>` 的函数体写法。
- 新增 `Bool`、`Option`、`Result`、right-biased `Either` 的 `then` / `else` selection、`_N` payload 占位符与 tuple group。
- 单臂缺失侧固定为 `Unit`；group subject 全部从左到右求值，group `else` 不暴露 payload。
- formatter 的单表达式 arm 不使用花括号，规范布局为缩进的 `then` / `else` 行。
- `if`、`if let`、`while let` 与完整 `match` 继续保留。
