# 0.4.0-alpha.1

- Replaced ability and handler examples with nominal `effect` tags and ordinary `need`-annotated functions.
- Documented `E: effects` higher-order polymorphism, qualified effect paths, independent function re-exports, and compile-time erasure.
- Removed handler, `with`, `resume`, CPS, runtime-dispatch, and effect-owned-operation material from the tutorial grammar and examples.
- Clarified that effect authorization does not grant borrow permissions.
- Established the first SemVer 2.0.0 Eidos language prerelease and its verified Eidosc compatibility range.
- Replaced `eidosVersion` and `[language].syntax` with `manifestSchema = 3` and `[language].version`.
