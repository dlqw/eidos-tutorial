# Eidos 0.5.0-alpha.1 typed CTFE and const generics

- Extended pure compile-time evaluation with typed String, tuple, list, ADT, recursive-call, pure-block, `if`, `match`, constructor-pattern, and field semantics.
- Added ordered type/value/effect-row generic argument domains and value-level const generics for ADTs, type aliases, constructors, functions, traits, implementations, and named instances.
- Made const values participate in nominal identity, specialization, layout, name mangling, coherence, cache fingerprints, payload round trips, and IDE semantic details.
- Added `examples/68_const_generics.eidos` and updated the English and Chinese tutorial grammar for Eidos `0.5.0-alpha.1`.
